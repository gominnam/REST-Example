## 학습으로 얻는 것 또는 이유
```
1. REST API에 아키텍쳐 이해
2. 많은 기업들이 사용하는 기술(대중적)
3. HTTP에 대한 이론적인 개념을 직접적인 이해
4. 배움으로 부터 오는 자신감
5. 개인적인 즐거움
```
